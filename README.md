# VideoLAN Python Media Player Demo

This is a sample Python VLC media player demo for GSoC.

## What it does
- Plays a local media file (audio or video)
- Demonstrates Python integration with VLC

## Tech used
- Python
- python-vlc

## How to run
1. Install Python 3
2. Install VLC python library: pip install python-vlc
3. Place a media file in the project folder (example: sample.mp3)
4. Run: python vlc_demo.py
